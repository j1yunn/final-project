카페 익명 방명록 어플

- 카페, 성인 인증한 회원에 한해 술집(bar)까지 방명록[쪽지]를 남길 수 있는 기능
- 이용자가 남긴 쪽지는 익명으로 누적되어 해당 카페에 방문한 사람이 GPS 기반 인증을 받아 확인
- 1일 / 1장소 / 1회 쪽지를 남기는 것이 가능하며 다른 장소에도 쪽지를 남길 수 있음
- 예시: 여기는 ㅇㅇㅇ가 맛있어요 부터 다음주 ㅇ요일 ㅇ시 등으로 만남을 제의하는 쪽지를 남기는 것도 가능
- 쪽지가 쌓여있는 이른바 핫플레이스와 같은 곳도 지역마다 표시
- 가게와의 제휴를 통해서 광고, 홍보 역시 가능
- 수익 구조 ... 고민해봐야 함
