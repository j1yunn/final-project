OTT 서비스 가격 종합해서 싸게 올리는 웹사이트

이유: 신청하는 사람들 한에서 랜덤으로 사람 매칭해줌으로써  OTT 서비스를 기존 OTT 구독 사이트보다 싸게, 종합된 가격을 구독형으로 제안함으로써 원하는 OTT 작품, 원하는 OTT 사이트를 추천해주고 정보 공유가 가능하게 한다.

CRUD

 특정 OTT 구인 게시판을 만들어서 특정  OTT만 원하는 사람들끼리 투표 형식이나 다른 형식으로 참여해서 랜덤 매칭이나 특정 사람들로 구독형으로 할 수 있게 제안

정보 게시판 생성 각 OTT별 최신작, 정주행 가이드 추천 등을 볼 수 있게 제안 (글 작성, 수정, 삭제, 댓글 달기, 추천)

- 부가적으로 30추 게시판, 10추 게시판을 만들어서 핫한 게시글을 볼 수 있는 게시판 제안

신고 게시판 생성 노쇼 등 각종 사건 사고를 게시할 수 있게 작성 ( 익명으로 작성되게, 비밀 댓글, 공개 댓글 기능 설정 제안)

건의 게시판 생성 말 그대로 건의를 할 수 있게 하는 게시판

로그인

SNS 로그인/ 일반 로그인 회원가입 로그아웃 기능 

가입 할 때 선호하는 장르 선정 - 차후 API를 활용 추천 OTT 작품이나 OTT 사이트 추천

회원 정보 수정에서 선호 장르 수정 가능

API

Chat GPT API를 활용 선호하는 장르 체크한 데이터 토대로 추천 OTT 작품이나 OTT 사이트 제안

세계관 용어나 각종 언어가 이해가 가지 않을 때, 사이트 정보 게시판에 있는 게시글 토대로 정보 제안 또는 Chat GPT API를 활용 정보를 알 수  있게 제안
